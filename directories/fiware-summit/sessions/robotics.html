FIWARE Global Summit: Robotics

Tech & Trends -

Join the Robotics Session at the FIWARE Global Summit 2024 for an in-depth exploration of how FIWARE architectures seamlessly integrate with two prominent robotics middleware technologies: OPC UA and ROS 2. This session will provide valuable insights into the latest advancements and practical applications of these integrations. The session is ideal for anyone interested in the intersection of robotics systems, middleware technologies, and IIoT applications.

Key Topics:

  • OPC UA Integration: Discover how the FIWARE IoT Agent technology facilitates the integration with OPC UA. Representatives from Engineering Ingegneria Informatica will showcase the current status and roadmap of the OPC UA Agent along with their IoT agent User Interface, a brand new feature designed to accelerate FIWARE IoT Agent management for robotics system administrators.
  • ROS 2 Integration: Learn about the seamless and real-time enabled integration of FIWARE with ROS 2 robots through an enhanced version of the Orion-LD context broker. A representative from Eprosima will present the current status and roadmap for this feature, along with Vulcanexus, a comprehensive ROS 2 toolset developed and maintained by Eprosima to overcome some limitations of the official ROS 2 distribution in commercial robotics solutions.
