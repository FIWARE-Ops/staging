FIWARE Webinar: FIWARE Data Space Technologies Webinar #6: Reference Use Case

Are you ready to deep dive into the world of data spaces and learn how FIWARE can improve your organization’s data-sharing strategies? This six-part webinar series will cover everything you need to know about FIWARE Data Space Technologies. This series is organized by the FIWARE Technical Steering Committee, with support from FIWARE Foundation.

Identifying the roles of the participants within a data space can be difficult, as participants can frequently undertake more than one role. To help understand these complexities, this week’s FIWARE Data Space Technologies webinar will dive into a reference use case detailing the participants’ descriptions and conclude with a deployment aligning to the business needs of the use case.
