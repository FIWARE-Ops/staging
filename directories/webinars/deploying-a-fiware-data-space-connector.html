FIWARE Webinars: Deploying a FIWARE Data Space Connector

A data space connector needs to fulfill many different roles to work successfully.

A data space connector needs to fulfill many different roles to work successfully. The FIWARE Data Space connector does this using the app-of-apps Helm chart paradigm, which allows for flexibility when deciding which underlying components to use.

This week, Stefan Wiedemann will explain the connector’s architecture and the details of each component’s role. Furthermore, deployments will be discussed, along with a simple demonstration of the components in use.
