FIWARE Global Summit: Digital Twin basic programming using NGSI-LD: CRUD, Registrations, Subscriptions

Tech Training - NGSI-LD Basics

Basic introduction to the core functionality of the NGSI-LD API and the link between NGSI-LD and JSON-LD. The session will cover core and user @context, basic CRUD actions, subscriptions and all four types of registration (inclusive, exclusive, auxilliary and redirect). The emphasis will be on designing NGSI-LD systems using Smart Data Models and creating an interface which will allow for data exchange across data spaces.
