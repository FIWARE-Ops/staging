FIWARE Global Summit: Introduction to JSON-LD and Linked Data

Tech Training - NGSI-LD Basics

This session will describe the prerequistes to understanding the NGSI-LD data format. Learn about Linked Data and the differences between JSON-LD and the JSON data format. The session will describe the structure of an @context mapping file and how to define user context within an NGSI-LD system.

This session is necessary to fully understand the structure of the API calls used in NGSI-LD.
